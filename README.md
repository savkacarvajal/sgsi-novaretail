# SGSI — NovaRetail SpA

Sistema de Gestión de Seguridad de la Información (SGSI) desarrollado por **Thynk Unlimited** para la empresa ficticia **NovaRetail SpA**, en el marco del ramo de ciberseguridad.

El SGSI está alineado con **ISO/IEC 27001:2022**, **PCI DSS** y la normativa chilena **Ley 21.663 (Marco de Ciberseguridad)** y **Ley 19.628 (Protección de Datos Personales)**.
